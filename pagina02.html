Os requisitos para jogar Fortnite

O Fortnite foi feito para rodar em computadores com uma grande gama de configurações diferentes, assim muitas pessoas podem subir no Ônibus de Batalha e jogar. Para ter a melhor experiência com Fortnite no PC, recomendamos as seguintes especificações de sistema.

Primeiramente teremos os requisitos mínimos:

  • Placa de Vídeo: Intel HD 4000 no PC; AMD Radeon Vega 8
  • Processador: Core i3-3225 3.3 GHz
  • Memória: 8 GB RAM
  • Sistema Operacional: Windows 10 64 bits versão 1703 ou Mac OS Mojave 10.14.6

Agora teremos os requisitos recomendados:

  • Placa de Vídeo: Nvidia GTX 960, AMD R9 280, ou placa equivalente compatível com DX11
  • Memória de Vídeo: 2 GB VRAM
  • Processador: Core i5-7300U 3.5 GHz, AMD Ryzen 3 3300U, ou equivalente
  • Memória: 16 GB RAM ou mais
  • Hard Drive: NVMe Solid State Drive
  • Sistema Operacional: Windows 10/11 64 bits

Para rodar o Fornite no épico, é necessário:

  • Placa de Vídeo: Nvidia RTX 3070, AMD Radeon RX 6700 XT, ou equivalente
  • Memória de Vídeo: 8 GB VRAM ou mais
  • Processador: Intel Core i7-8700, AMD Ryzen 7 3700x ou equivalente
  • Memória: 16 GB RAM ou mais
  • Hard Drive: NVMe Solid State Drive
  • Sistema Operacional: Windows 10/11 64 bits
